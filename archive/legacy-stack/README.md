# Legacy Stack Archive

This folder contains the retired pre-`lumino-web` codebase that used to live at the repository root.

It is archived here so:

- the active app stays visually and structurally centered on `lumino-web`
- old deployment entrypoints do not keep competing with the current app
- the legacy code remains recoverable until you decide to delete it

If you later want to remove the legacy stack completely, this is the folder to delete.
